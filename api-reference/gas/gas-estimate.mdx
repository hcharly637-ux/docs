title: "Estimate Transaction Confirmation Time by Gas Price"
sidebarTitle: "Gas Estimate"
description: "GET /v2/api?module=gastracker&action=gasestimate — Estimates the confirmation time in seconds for a transaction given a specified gas price in Wei."
---

The Gas Estimate endpoint allows you to predict how long a transaction will take to be confirmed on the network based on the gas price you plan to offer. You provide a gas price in Wei, and the API returns the estimated wait time in seconds. This is useful for building dynamic fee UI components or deciding between competing gas price tiers at submission time.

## Query Parameters

<ParamField query="apikey" type="string" required>
  Your Etherscan API key. You can obtain one from the [Etherscan API dashboard](https://etherscan.io/myapikey).
</ParamField>

<ParamField query="chainid" type="string" default="1">
  Chain ID to target. Use `1` for Ethereum Mainnet. See the full list of [supported chains](/supported-chains).
</ParamField>

<ParamField query="module" type="string" required>
  Set to `gastracker` for this endpoint.
</ParamField>

<ParamField query="action" type="string" required>
  Set to `gasestimate` for this endpoint.
</ParamField>

<ParamField query="gasprice" type="string" required>
  The gas price you intend to pay, expressed in Wei. For example, `2000000000` equals 2 Gwei.
</ParamField>

## Response Fields

<ResponseField name="status" type="string">
  `"1"` indicates a successful response; `"0"` indicates an error.
</ResponseField>

<ResponseField name="message" type="string">
  A short status message such as `"OK"`.
</ResponseField>

<ResponseField name="result" type="string">
  The estimated confirmation time in seconds for a transaction submitted at the specified gas price.
</ResponseField>

## Example Request

```bash
curl "https://api.etherscan.io/v2/api?chainid=1&module=gastracker&action=gasestimate&gasprice=2000000000&apikey=YourApiKeyToken"
```

## Example Response

```json
{
  "status": "1",
  "message": "OK",
  "result": "45"
}
```
