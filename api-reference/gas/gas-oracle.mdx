The Gas Oracle endpoint returns real-time gas price recommendations derived from recent block activity on the network. You can use these values to decide which gas price tier fits your transaction's urgency — `SafeGasPrice` for low-priority submissions, `ProposeGasPrice` for standard inclusion, or `FastGasPrice` when speed matters most. All prices are returned in Gwei.

## Query Parameters

<ParamField query="apikey" type="string" required>
  Your Etherscan API key. You can obtain one from the [Etherscan API dashboard](https://etherscan.io/myapikey).
</ParamField>

<ParamField query="chainid" type="string" default="1">
  Chain ID to target. Use `1` for Ethereum Mainnet. See the full list of [supported chains](/supported-chains).
</ParamField>

<ParamField query="module" type="string" required>
  Set to `gastracker` for this endpoint.
</ParamField>

<ParamField query="action" type="string" required>
  Set to `gasoracle` for this endpoint.
</ParamField>

## Response Fields

<ResponseField name="status" type="string">
  `"1"` indicates a successful response; `"0"` indicates an error.
</ResponseField>

<ResponseField name="message" type="string">
  A short status message such as `"OK"`.
</ResponseField>

<ResponseField name="result" type="object">
  An object containing the current gas price recommendations.

  <Expandable title="result fields">
    <ResponseField name="LastBlock" type="string">
      The block number used to calculate the gas price recommendations.
    </ResponseField>
    <ResponseField name="SafeGasPrice" type="string">
      A gas price (in Gwei) that is likely to be included in the next few blocks without urgency.
    </ResponseField>
    <ResponseField name="ProposeGasPrice" type="string">
      A gas price (in Gwei) recommended for standard transaction inclusion.
    </ResponseField>
    <ResponseField name="FastGasPrice" type="string">
      A gas price (in Gwei) for fast inclusion, typically within the next block.
    </ResponseField>
    <ResponseField name="suggestBaseFee" type="string">
      The current base fee per gas in Gwei, as defined by EIP-1559.
    </ResponseField>
    <ResponseField name="gasUsedRatio" type="string">
      A comma-separated list of gas used ratios for the last several blocks, expressed as a fraction of the gas limit.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl "https://api.etherscan.io/v2/api?chainid=1&module=gastracker&action=gasoracle&apikey=YourApiKeyToken"
```

## Supported Networks

This endpoint is available on Ethereum Mainnet, BNB Smart Chain Mainnet, Polygon Mainnet, Moonbeam Mainnet, Moonriver Mainnet, Celo Mainnet, Gnosis, and Linea Mainnet.
