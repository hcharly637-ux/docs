These PRO endpoints give you a comprehensive historical view of Ethereum network activity and health over time. You can query daily transaction volumes, new address creation rates, block space utilization, mining power, proof-of-work difficulty, and the total fees paid to validators — all scoped to any date range you choose. Each endpoint shares the same date-range and sort parameters for consistency.

<Warning>
  All endpoints on this page are **PRO** and require a [Standard Plan](/resources/rate-limits) or above to access.
</Warning>

---

## Daily Transaction Count

Retrieves the number of transactions processed on the network for each day in a given date range. Use this to track adoption trends, identify high-traffic periods, and correlate network activity with market events.

### Query Parameters

<ParamField query="apikey" type="string" required>
  Your Etherscan API key.
</ParamField>

<ParamField query="chainid" type="string" default="1">
  Chain ID to target. Use `1` for Ethereum Mainnet. See the full list of [supported chains](/supported-chains).
</ParamField>

<ParamField query="module" type="string" required>
  Set to `stats` for this endpoint.
</ParamField>

<ParamField query="action" type="string" required>
  Set to `dailytx` for this endpoint.
</ParamField>

<ParamField query="startdate" type="string" required>
  Start of the date range in `yyyy-MM-dd` format, e.g. `2019-02-01`.
</ParamField>

<ParamField query="enddate" type="string" required>
  End of the date range in `yyyy-MM-dd` format, e.g. `2019-02-28`.
</ParamField>

<ParamField query="sort" type="string" default="asc">
  Sort order: `asc` for oldest-first or `desc` for newest-first.
</ParamField>

### Response Fields

<ResponseField name="result" type="array">
  An array of daily transaction count records.

  <Expandable title="result item fields">
    <ResponseField name="UTCDate" type="string">
      The date of the record in `yyyy-MM-dd` format.
    </ResponseField>
    <ResponseField name="unixTimeStamp" type="string">
      Unix timestamp for the start of the UTC day.
    </ResponseField>
    <ResponseField name="transactionCount" type="number">
      Total number of transactions processed on that day.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl "https://api.etherscan.io/v2/api?chainid=1&module=stats&action=dailytx&startdate=2019-02-01&enddate=2019-02-28&sort=asc&apikey=YourApiKeyToken"
```

### Example Response

```json
{
  "status": "1",
  "message": "OK",
  "result": [
    {
      "UTCDate": "2019-02-01",
      "unixTimeStamp": "1548979200",
      "transactionCount": 498856
    },
    {
      "UTCDate": "2019-02-02",
      "unixTimeStamp": "1549065600",
      "transactionCount": 450314
    }
  ]
}
```
