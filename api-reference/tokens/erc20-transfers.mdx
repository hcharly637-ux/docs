Use this endpoint to retrieve the full ERC-20 token transfer history for a given address. You can optionally narrow results to a specific token contract or a block range, and paginate through large result sets using the `page` and `offset` parameters.

<Warning>
  **Effective July 1, 2026**, the maximum records returned per request will be reduced from 10,000 to 1,000 for Free tier API users. See the full list of [affected endpoints](https://docs.etherscan.io/changelog#upcoming-change-reduced-maximum-records-per-request-on-the-free-api-tier).
</Warning>

## Query Parameters

<ParamField query="apikey" type="string" default="YourApiKeyToken">
  Your Etherscan API key.
</ParamField>

<ParamField query="chainid" type="string" default="1">
  Chain ID to query, e.g. `1` for Ethereum, `8453` for Base from our [supported chains](/supported-chains).
</ParamField>

<ParamField query="module" type="string" default="account">
  Set to `account` for this endpoint.
</ParamField>

<ParamField query="action" type="string" default="tokentx">
  Set to `tokentx` for this endpoint.
</ParamField>

<ParamField query="contractaddress" type="string" default="0x9f8f72aa9304c8b593d555f12ef6589cc3a579a2">
  The ERC-20 token contract address to filter transfers by, e.g. `0xdac17f958d2ee523a2206206994597c13d831ec7` for USDT.
</ParamField>

<ParamField query="address" type="string" default="0x4e83362442b8d1bec281594cea3050c8eb01311c">
  The address to query, like `0xfefefefefefefefefefefefefefefefefefefefe`.
</ParamField>

<ParamField query="startblock" type="integer" default="0">
  Starting block number to search from.
</ParamField>

<ParamField query="endblock" type="integer" default="999999999">
  Ending block number to search to.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number for pagination.
</ParamField>

<ParamField query="offset" type="integer" default="100">
  Number of records per page. Use the `page` parameter for subsequent records.
</ParamField>

<ParamField query="sort" type="string" default="asc">
  Sort order: `desc` for the latest transactions first, or `asc` for the oldest transactions first.
</ParamField>

## Example Request

```bash
curl "https://api.etherscan.io/v2/api\
?chainid=1\
&module=account\
&action=tokentx\
&contractaddress=0x9f8f72aa9304c8b593d555f12ef6589cc3a579a2\
&address=0x4e83362442b8d1bec281594cea3050c8eb01311c\
&startblock=0\
&endblock=999999999\
&page=1\
&offset=100\
&sort=asc\
&apikey=YourApiKeyToken"
```

## Response

The `result` array contains one object per ERC-20 transfer event.

<ResponseField name="status" type="string">
  `"1"` on success, `"0"` on failure.
</ResponseField>

<ResponseField name="message" type="string">
  `"OK"` on success, or an error description on failure.
</ResponseField>

<ResponseField name="result" type="array">
  Array of ERC-20 transfer objects.

  <ResponseField name="blockNumber" type="string">
    Block number in which the transfer was included.
  </ResponseField>

  <ResponseField name="timeStamp" type="string">
    Unix timestamp of the block.
  </ResponseField>

  <ResponseField name="hash" type="string">
    Transaction hash.
  </ResponseField>

  <ResponseField name="nonce" type="string">
    Sender's transaction count at the time of this transaction.
  </ResponseField>

  <ResponseField name="blockHash" type="string">
    Hash of the block containing this transaction.
  </ResponseField>

  <ResponseField name="from" type="string">
    Sender address of the token transfer.
  </ResponseField>

  <ResponseField name="contractAddress" type="string">
    Token contract address.
  </ResponseField>

  <ResponseField name="to" type="string">
    Recipient address of the token transfer.
  </ResponseField>

  <ResponseField name="value" type="string">
    Amount transferred, in the token's smallest unit (apply `tokenDecimal` to convert to a human-readable value).
  </ResponseField>

  <ResponseField name="tokenName" type="string">
    Full name of the token, e.g. `"Maker"`.
  </ResponseField>

  <ResponseField name="tokenSymbol" type="string">
    Token ticker symbol, e.g. `"MKR"`.
  </ResponseField>

  <ResponseField name="tokenDecimal" type="string">
    Number of decimal places used by the token.
  </ResponseField>

  <ResponseField name="transactionIndex" type="string">
    Position of the transaction within the block.
  </ResponseField>

  <ResponseField name="gas" type="string">
    Gas limit set for the transaction.
  </ResponseField>

  <ResponseField name="gasPrice" type="string">
    Gas price in wei.
  </ResponseField>

  <ResponseField name="gasUsed" type="string">
    Actual gas consumed by the transaction.
  </ResponseField>

  <ResponseField name="cumulativeGasUsed" type="string">
    Total gas used in the block up to and including this transaction.
  </ResponseField>

  <ResponseField name="input" type="string">
    Deprecated. Always returns `"deprecated"`.
  </ResponseField>

  <ResponseField name="methodId" type="string">
    First four bytes of the call data, identifying the function invoked.
  </ResponseField>

  <ResponseField name="functionName" type="string">
    Human-readable name of the function called, if available.
  </ResponseField>

  <ResponseField name="confirmations" type="string">
    Number of block confirmations since this transaction was mined.
  </ResponseField>
</ResponseField>
