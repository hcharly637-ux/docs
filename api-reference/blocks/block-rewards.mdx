---
title: "Get Block and Uncle Rewards by Block Number"
sidebarTitle: "Block Rewards"
description: "GET /v2/api?module=block&action=getblockreward — Retrieves block mining rewards and associated uncle block rewards for a specific block number."
---

Every mined Ethereum block generates a block reward for the miner, and may also include uncle inclusion rewards when valid uncle blocks are referenced. This endpoint returns the full reward breakdown for a specific block: the miner's address, the base block reward, the details of any uncle blocks (including each uncle miner's reward), and the uncle inclusion bonus paid to the main block miner. All reward values are returned in Wei.

## Query Parameters

<ParamField query="chainid" type="string" default="1" required>
  Chain ID to query, such as `1` for Ethereum or `8453` for Base. See [supported chains](/supported-chains) for the full list.
</ParamField>

<ParamField query="module" type="string" default="block" required>
  Set to `block` for this endpoint.
</ParamField>

<ParamField query="action" type="string" default="getblockreward" required>
  Set to `getblockreward` for this endpoint.
</ParamField>

<ParamField query="blockno" type="integer" default="2165403" required>
  The block number to retrieve reward information for.
</ParamField>

<ParamField query="apikey" type="string" default="YourApiKeyToken" required>
  Your Etherscan API key.
</ParamField>

## Response Fields

<ResponseField name="status" type="string">
  Returns `1` when the request succeeds, or `0` on failure.
</ResponseField>

<ResponseField name="message" type="string">
  Returns `OK` on success, or an error description on failure.
</ResponseField>

<ResponseField name="result" type="object">
  An object containing the block reward details.

  <Expandable title="result properties">
    <ResponseField name="blockNumber" type="string">
      The queried block number.
    </ResponseField>

    <ResponseField name="timeStamp" type="string">
      Unix timestamp of when the block was mined.
    </ResponseField>

    <ResponseField name="blockMiner" type="string">
      The address of the miner who produced this block.
    </ResponseField>

    <ResponseField name="blockReward" type="string">
      The total block reward in Wei, including transaction fees and uncle inclusion rewards.
    </ResponseField>

    <ResponseField name="uncles" type="array">
      An array of uncle block objects included in this block.

      <Expandable title="uncle properties">
        <ResponseField name="miner" type="string">
          The address of the uncle block's miner.
        </ResponseField>

        <ResponseField name="unclePosition" type="string">
          The position (index) of the uncle block within the block's uncle list.
        </ResponseField>

        <ResponseField name="blockreward" type="string">
          The reward paid to the uncle block miner in Wei.
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="uncleInclusionReward" type="string">
      The additional reward paid to the main block miner for including uncle blocks, in Wei.
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl "https://api.etherscan.io/v2/api
  ?chainid=1
  &module=block
  &action=getblockreward
  &blockno=2165403
  &apikey=YourApiKeyToken"
```
