This page consolidates the daily block analytics endpoints, giving you access to historical blockchain health metrics over any date range. You can query the average size of blocks mined each day, how many blocks were produced, total rewards distributed, average block times, and uncle block statistics. All endpoints use `yyyy-MM-dd` date strings and support ascending or descending sort order.

<Warning>
  All endpoints on this page are **PRO endpoints** available to the Standard plan and above. Calling these endpoints with a free API key will return an error.
</Warning>

---

## Get Daily Average Block Size

Retrieves the daily average block size in bytes over a specified date range. Tracking block size over time helps you understand network load trends and how much data the chain is processing per block.

### Query Parameters

<ParamField query="chainid" type="string" default="1" required>
  Chain ID to query, such as `1` for Ethereum or `8453` for Base. See [supported chains](/supported-chains) for the full list.
</ParamField>

<ParamField query="module" type="string" default="stats" required>
  Set to `stats` for this endpoint.
</ParamField>

<ParamField query="action" type="string" default="dailyavgblocksize" required>
  Set to `dailyavgblocksize` for this endpoint.
</ParamField>

<ParamField query="startdate" type="string" default="2019-02-01" required>
  Starting date in `yyyy-MM-dd` format.
</ParamField>

<ParamField query="enddate" type="string" default="2019-02-28" required>
  Ending date in `yyyy-MM-dd` format.
</ParamField>

<ParamField query="sort" type="string" default="asc">
  Sort order, either `asc` for oldest-first or `desc` for newest-first.
</ParamField>

<ParamField query="apikey" type="string" default="YourApiKeyToken" required>
  Your Etherscan API key.
</ParamField>

### Response Fields

<ResponseField name="status" type="string">
  Returns `1` when the request succeeds, or `0` on failure.
</ResponseField>

<ResponseField name="message" type="string">
  Returns `OK` on success, or an error description on failure.
</ResponseField>

<ResponseField name="result" type="array">
  An array of daily average block size records.

  <Expandable title="result item properties">
    <ResponseField name="UTCDate" type="string">
      The date in `yyyy-MM-dd` format (UTC).
    </ResponseField>

    <ResponseField name="unixTimeStamp" type="string">
      Unix timestamp for the start of that UTC day.
    </ResponseField>

    <ResponseField name="blockSize_bytes" type="integer">
      The average block size in bytes for that day.
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

```bash
curl "https://api.etherscan.io/v2/api
  ?chainid=1
  &module=stats
  &action=dailyavgblocksize
  &startdate=2019-02-01
  &enddate=2019-02-28
  &sort=asc
  &apikey=YourApiKeyToken"
```

### Example Response

```json
{
  "status": "1",
  "message": "OK",
  "result": [
    {
      "UTCDate": "2019-02-01",
      "unixTimeStamp": "1548979200",
      "blockSize_bytes": 20373
    },
    {
      "UTCDate": "2019-02-02",
      "unixTimeStamp": "1549065600",
      "blockSize_bytes": 17499
    }
  ]
}
```

## Get Daily Uncle Block Count and Rewards

Miners who produce uncles still receive a partial reward. Tracking uncle rates can provide insight into network propagation latency and mining pool distribution.
