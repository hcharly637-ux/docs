title: "Get Native Token Balance for Up to 20 Wallet Addresses"

---

The `balancemulti` action retrieves the current native token balance for up to 20 addresses in a single API call, making it significantly more efficient than issuing individual balance requests. Balances are returned in the chain's smallest denomination (e.g. wei for ETH). Addresses are passed as a comma-separated list in the `address` parameter.

## Request

```bash
curl "https://api.etherscan.io/v2/api\
?chainid=1\
&module=account\
&action=balancemulti\
&address=0xddbd2b932c763ba5b1b7ae3b362eac3e8d40121a,0x63a9975ba31b0b9626b34300f7f627147df1f526,0x198ef1ec325a96cc354c7266a038be8b5c558f67\
&tag=latest\
&apikey=YourApiKeyToken"
```

## Query Parameters

<ParamField query="apikey" type="string" default="YourApiKeyToken" required>
  Your Etherscan API key.
</ParamField>

<ParamField query="chainid" type="string" default="1" required>
  Chain ID to query, e.g. `1` for Ethereum, `8453` for Base. See the full list at [supported chains](/api-reference/chainlist).
</ParamField>

<ParamField query="module" type="string" default="account" required>
  Set to `account` for this endpoint.
</ParamField>

<ParamField query="action" type="string" default="balancemulti" required>
  Set to `balancemulti` for this endpoint.
</ParamField>

<ParamField query="address" type="string" default="0xddbd2b932c763ba5b1b7ae3b362eac3e8d40121a,0x63a9975ba31b0b9626b34300f7f627147df1f526,0x198ef1ec325a96cc354c7266a038be8b5c558f67" required>
  A comma-separated list of addresses to query. You may include up to a maximum of **20 addresses** per call.
</ParamField>

<ParamField query="tag" type="string" default="latest">
  Use `latest` for balances at the most recent block. Also accepts a specific block number in hex format (e.g. `0x10d4f`) for balances up to the last 128 blocks.
</ParamField>

## Response Fields

<ResponseField name="status" type="string">
  `"1"` on success, `"0"` on error.
</ResponseField>

<ResponseField name="message" type="string">
  `"OK"` on success, or an error message string.
</ResponseField>

<ResponseField name="result" type="array">
  An array of balance objects, one per queried address. The order matches the order of addresses supplied in the request.
</ResponseField>

<ResponseField name="result[].account" type="string">
  The queried address.
</ResponseField>

<ResponseField name="result[].balance" type="string">
  The native token balance of the address in the chain's smallest unit (wei for ETH). Divide by `10^18` to convert to ETH.
</ResponseField>
