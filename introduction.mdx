Etherscan is the leading blockchain explorer, search, API, and analytics platform for Ethereum and other EVM-compatible chains. Whether you're building a DeFi dashboard, a wallet, an on-chain analytics tool, or a smart contract integration, Etherscan gives you reliable, structured access to the blockchain data your application needs.

## What You Can Build

With the Etherscan API, you can query on-chain data across all supported networks — no node required. Use it to retrieve wallet balances, trace token transfers, decode contract ABIs, fetch event logs, monitor gas prices, and run deep analytics across the full history of any EVM chain.

Key capabilities include:

- **Accounts** — Query native balances, transaction histories, and internal transactions for any address
- **Tokens** — Fetch ERC-20, ERC-721, and ERC-1155 transfers, holder lists, and token balances
- **Contracts** — Retrieve verified source code, ABIs, and contract creation metadata
- **Logs** — Filter event logs by address, topic, and block range with precision
- **Gas** — Monitor real-time gas prices and historical gas usage trends
- **Analytics** — Access daily network stats, block rewards, new address counts, and more

## API V2: One Key for Every Chain

With **Etherscan API V2**, all 60+ [supported chains](/supported-chains) are unified under a single account and a single API key. Your app becomes multichain simply by changing the `chainid` parameter — no new accounts, no extra keys, no per-chain dashboards.

```
https://api.etherscan.io/v2/api?chainid=1&module=account&action=balance&...
```

Switch `chainid=1` to `chainid=56` for BNB Smart Chain, `chainid=8453` for Base, `chainid=42161` for Arbitrum One, and so on. Every endpoint, every chain, one unified key.

<Note>
  New API keys activate within a few minutes and are immediately valid across all 60+ supported chains.
</Note>
