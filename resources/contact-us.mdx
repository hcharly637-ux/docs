The Etherscan API support team is available to help you with integration questions, error troubleshooting, plan upgrades, and custom or enterprise data requirements. Reach out through the appropriate channel below depending on your need.

## API Support

For technical questions about the API — including error messages, endpoint behavior, rate limit concerns, or integration help — email the API support team directly:

**[apisupport@etherscan.io](mailto:apisupport@etherscan.io)**

For dedicated or custom API plans with higher rate limits or specialized data access, contact API support to discuss your requirements.

<Tip>
  When reaching out, include the following to help the team resolve your issue faster:
  - Your API key (mask the middle characters, e.g., `ABCD****WXYZ`)
  - The exact error message returned in the `result` field
  - The full request URL (with your key masked)
  - The chain ID you're querying
</Tip>

## General Inquiries

For account issues, billing questions, or general product feedback, visit the Etherscan contact page:

**[etherscan.io/contactus](https://etherscan.io/contactus)**
